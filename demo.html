jquery.charPicker

Вставка спецсимволов и эмодзи в input/textarea. Ленивая загрузка ядра. Поиск с коррекцией раскладки EN↔RU и ё/е. Клавиатурная навигация. Позиционирование по якорю.

01

Базовый — trigger:'auto'

Режим по умолчанию. Плагин создаёт кнопку-триггер рядом с полем. Ядро (charPicker.core.js + charPicker.core.css) загружается лениво — только при первом клике.

ℹ️
Локальное открытие (file://): браузер блокирует XHR-запросы, поэтому charpicker-data.min.json не загружается. Плагин автоматически переключается на встроенный тестовый набор (~197 символов). Для полного набора откройте страницу через веб-сервер.
📝
input[type=text]
Все группы символов
— ожидание выбора —
📄
textarea
Вставка в позицию курсора
— ожидание выбора —
JavaScript
// Минимальная инициализация — кнопка создаётся автоматически
$('#my-field').charPicker({
  dataUrl: '/js/charpicker-data.min.json'
});

03

Внешний триггер — trigger: Element

Передайте готовый DOM-элемент или jQuery-объект как trigger. Плагин сам повесит click и будет позиционировать попап относительно этой кнопки.

🔗
trigger: $('#my-btn')
Попап под переданной кнопкой
— ожидание выбора —
🔗
trigger: false + вызов через API
Полный контроль — когда открывать
— ожидание выбора —
JavaScript
// Вариант А: передаём готовую кнопку
$('#my-field').charPicker({
  trigger: $('#my-btn'),  // плагин вешает click сам
  dataUrl: '/js/data.json'
});

// Вариант Б: trigger:false — открытие только через API
$('#my-field').charPicker({
  trigger: false,
  dataUrl: '/js/data.json'
});
// Открываем кнопкой, передаём её как якорь позиционирования:
$('#my-btn').on('click', function() {
  $('#my-field').charPicker('open', this);
});

Встройка в тулбар — trigger: $('.charpicker')

Типичный сценарий для форм с тулбаром. Кнопка уже в HTML, плагин вешает на неё обработчик и позиционирует попап под ней.

— ожидание выбора —
HTML + JavaScript
<!-- HTML: кнопка в тулбаре -->
<div class="toolbar">
  <button class="toolbar-btn charpicker">☺</button>
</div>
<textarea id="my-field"></textarea>

// JS: передаём кнопку из тулбара как триггер
$('#my-field').charPicker({
  trigger: $('.charpicker'),
  dataUrl: '/js/data.json'
});
05

Колбэк trigger — полный контроль

Для сложных случаев: кнопка создаётся или подключается через колбэк. Колбэк получает функцию open(anchor) — вызовите её когда нужно. this внутри колбэка — DOM-элемент поля ввода.

🔧
trigger: function(open) { … }
Кнопка добавляется через колбэк
— ожидание выбора —
JavaScript
$('#my-field').charPicker({
  trigger: function(open) {
    // this = DOM textarea
    var $btn = $('<button type="button">☺ Символ</button>');
    $btn.on('click', function(e) {
      open(e.currentTarget); // передаём кнопку как якорь
    });
    $(this).closest('.form').find('.toolbar').append($btn);
  },
  dataUrl: '/js/data.json'
});

08

Управление через API — колбэки и методы

Публичные методы: open, close, toggle, destroy. Колбэки: onOpen, onClose, onSelect, onDataError.

🎛
Методы и колбэки
open / close / toggle / destroy + reinit
— ожидание события —
JavaScript
$('#my-field').charPicker({
  dataUrl: '/js/data.json',
  onOpen:  function() { console.log('opened'); },
  onClose: function() { console.log('closed'); },
  onSelect: function(char, name) {
    console.log('picked', char, name);
  },
  onDataError: function(err) {
    console.error('data error', err);
  }
});

// Методы
$('#my-field').charPicker('open');        // открыть
$('#my-field').charPicker('open', btn); // открыть с якорем
$('#my-field').charPicker('close');
$('#my-field').charPicker('toggle');
$('#my-field').charPicker('destroy');

DOM-структура

ℹ️
Ядро (charPicker.core.js) не создаёт wrap вокруг поля и не трогает DOM снаружи попапа. В режиме trigger:'auto' wrap и кнопку создаёт jquery.charPicker.v3.js. Попап (.jcp__popup) добавляется в <body> с position:fixed.

trigger:'auto' — поле обёртывается в .jcp

<!-- jquery.charPicker.v3.js оборачивает поле --> <span class="jcp"> <input type="text" class="charPicker"> <button class="jcp__trigger" aria-haspopup="dialog">…svg…</button> </span>

trigger: Element — wrap не создаётся

<!-- Ваша кнопка остаётся где была --> <button id="my-btn" aria-haspopup="dialog" aria-expanded="false">☺</button> <textarea id="my-field"></textarea>

Попап в <body> — position:fixed

<div class="jcp__popup jcp__popup--pos-bottom" role="dialog" aria-modal="true"> <div class="jcp__search"> <input class="jcp__search-input" type="search" placeholder="Поиск…"> </div> <div class="jcp__tabs" role="tablist"> <div class="jcp__tab jcp__tab--active" data-group="smileys">😀</div> … </div> <div class="jcp__body"> <div class="jcp__subgroup"> <div class="jcp__subgroup-title">Счастье</div> <div class="jcp__grid"> <div class="jcp__char" role="button" tabindex="0" data-char="😀" aria-label="улыбка grinning face">😀</div> … </div> </div> </div> </div> <!-- Тултип — один на страницу, position:fixed --> <div id="jcp-tooltip" class="jcp__tooltip" role="tooltip"></div>
