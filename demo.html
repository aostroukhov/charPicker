☺ jquery.charPicker

jQuery-плагин для вставки спецсимволов и эмодзи в input / textarea.
Эмулирует на десктопе функциональность мобильной клавиатуры.

v1.0.0 jQuery ≥ 1.9 Без зависимостей XSS-safe MIT

API — методы

Вызов Описание
$(el).charPicker() Инициализация с настройками по умолчанию
$(el).charPicker({ recentMax: 16 }) С опциями
$(el).charPicker('open') Открыть попап программно
$(el).charPicker('close') Закрыть попап
$(el).charPicker('destroy') Снести плагин, вернуть элемент в исходное состояние
